# Task: Setup wizard auto-redirect after gateway ready

## Description

Add JavaScript polling to the setup wizard frontend. After step 6 ("Switching network routes") completes, poll the gateway URL every 3 seconds. When the gateway responds with 200, auto-redirect to the OpenClaw Control UI with the gateway token.

## Acceptance Criteria
- [ ] After Cloudflare route switch, JS polling starts automatically
- [ ] Polls `https://<domain>/` every 3 seconds
- [ ] On 200 response, shows "系統就緒，正在跳轉..." for 2 seconds then redirects
- [ ] Redirect URL: `https://<domain>/#token=<gateway_token>`
- [ ] Fallback: after 60 seconds, shows manual link with the URL

## Technical Details

File: `openclaw-k3s-paas/setup-wizard/templates/index.html`

Add to the step 7 (deployment complete) handler or after step 6 success:
```javascript
function pollAndRedirect(domain, token) {
  const maxAttempts = 20; // 60s total
  let attempts = 0;
  const interval = setInterval(async () => {
    attempts++;
    try {
      const res = await fetch(`https://${domain}/`, { mode: 'no-cors' });
      // no-cors gives opaque response, but no error means reachable
      clearInterval(interval);
      statusEl.textContent = '系統就緒，正在跳轉...';
      setTimeout(() => {
        window.location.href = `https://${domain}/#token=${token}`;
      }, 2000);
    } catch (e) {
      if (attempts >= maxAttempts) {
        clearInterval(interval);
        statusEl.innerHTML = `<a href="https://${domain}/#token=${token}">點擊此處手動開啟 OpenClaw</a>`;
      }
    }
  }, 3000);
}
```
