# Task: Cloudflared graceful shutdown and Recreate strategy

## Description

Modify `04-cloudflared.yaml` to prevent ghost tunnel connections after pod restarts. Add preStop lifecycle hook, increase termination grace period, and switch deployment strategy to Recreate.

## Acceptance Criteria
- [ ] preStop hook added with `sleep 5` to allow graceful tunnel disconnection
- [ ] `terminationGracePeriodSeconds: 30` set on the pod spec
- [ ] Deployment strategy changed from RollingUpdate to Recreate
- [ ] After `kubectl rollout restart`, 100% of requests return 200 within 30s

## Technical Details

File: `openclaw-k3s-paas/k8s-manifests/04-cloudflared.yaml`

```yaml
spec:
  strategy:
    type: Recreate
  template:
    spec:
      terminationGracePeriodSeconds: 30
      containers:
        - name: cloudflared
          lifecycle:
            preStop:
              exec:
                command: ["sh", "-c", "sleep 5"]
```
